# 🤖 Dynamic Wumpus Logic Agent 

A Web-based AI Agent that navigates a dynamic Wumpus World using Knowledge-Based reasoning and Propositional Logic.


##  Project Overview

This project implements a **Dynamic Pathfinding Agent** that:

- Navigates a grid-based Wumpus World
- Uses **Breeze** and **Stench** percepts
- Maintains a **Knowledge Base (KB)**
- Uses logic inference to detect safe cells
- Runs in a **Flask Web App with real-time visualization**



##  AI Concepts Used

- Propositional Logic
- Knowledge Base (TELL / ASK model)
- Inference Engine
- Resolution-based reasoning (basic simulation)
- Percept-based decision making


##  Features

Dynamic Grid Size (Rows × Columns)  
Random Pit & Wumpus placement  
Real-time agent movement  
Breeze & Stench percepts  
Step-by-step simulation  
Web-based visualization (HTML + JS)  
REST API using Flask
